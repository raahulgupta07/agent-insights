Dropout rate · per district
approved
Share of calls that did NOT convert to an Ensure order.
SELECT district, COUNT(*) FILTER (WHERE call_outcome <> 'Ensure')::float / NULLIF(COUNT(*),0) AS dropout_rate FROM crm_calls GROUP BY district;
grain: district · unit: % · null districts excluded

“Top 10 districts by Ensure conversions this quarter”
verified
SELECT district, COUNT(*) AS conversions FROM crm_calls WHERE call_outcome = 'Ensure' AND call_date >= date_trunc('quarter', CURRENT_DATE) GROUP BY district ORDER BY conversions DESC LIMIT 10;
last run 2d ago · 1 row/district · used 4×
“Monthly call volume by channel”
verified
SELECT date_trunc('month', call_date) m, channel, COUNT(*) FROM crm_calls GROUP BY 1,2 ORDER BY 1;
