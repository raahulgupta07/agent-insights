Studio = the one user surface · Admin owns Connectors

Admin configures real data connections once. Users never touch DB credentials — they pick a ready source inside their Studio and build everything else (tables, instructions, persona, chat, skills, share) in one place.

ADMIN
1 · Configure connection
Manage → Connectors. Add DB + credentials. Provisions a shared, org-wide source. create_data_source permission.
→
USER
2 · Pick source in Studio
Studio → Sources → “Add source”. Choose from admin-provisioned sources. No creds, no DB setup.
→
USER
3 · Build the agent
Tables, instructions, persona, chat, skills, members — all inside the Studio. One surface, all features.
